    ディアボロとボールを主にやっています。ディアボロは3ディアの練習を、ボールは5ボールと4ボールの技数を増やすための練習をしています。

GitHubのリポジトリについて

SATySFi関係

  • exdesign :
    文書ごとにデザインを変更することを簡単にできるようにしたクラスファイルです。
  • stjarticle :
    SATySFiにもとからついているパッケージである、stdjaやstdjabookにいくつかのコマンドや機能を追加したものです。
  • musikui.satyh :
    SATySFiで虫食い算を描画するのに使えるパッケージです。
  • SATySFi-siunitx :
    SATySFiで国際単位系(SI)を使いたい時に、それをサポートするためのパッケージです。
  • SATySFi-uline :
    SATySFiで下線を引く時に便利なパッケージです。直線以外にも点線や打ち消し線もあります。
  • SATySFi-new-cls :
    改変を前提としたSATySFiのクラスファイル。やっていることの解説から改造の仕方までを説明する文書を公開するはず。
  • SATySFi-make-markdown :
    テキストモードを使ってsaty -> mdの変換をするためのクラスファイルとパッケージ。
  • SATySFi-debug-show-value :
    デバッグ用に使ってください。
  • SATySFi-clock-board :
    SATySFiで時計版、やってみたくなるよね。
  • SATySFi-calendar :
    SATySFiでカレンダー、やってみたくなるよね。

LaTeX関係

  • musikui.sty :
    掛け算とわり算の虫食い算をLaTeXで簡単に表現するためのパッケージです。CTANに収録され、TeXLive2018にも取り込まれています。
  • joubun :
    LaTeXで規則等を楽に作成するためのパッケージです。ある程度のところまでできています。
  • Jcounter :
    LaTeXで、日本で主に使われる数字の書き方に変換するコマンドを追加するパッケージです。完成まであと一歩、というところで止まっています。
